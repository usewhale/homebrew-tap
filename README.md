# Homebrew Tap for Whale

Install Whale with Homebrew:

```sh
brew install usewhale/tap/whale
```

Upgrade Whale:

```sh
brew upgrade whale
```

Whale is a DeepSeek-native coding agent for the terminal.
See the main project at https://github.com/usewhale/whale.

## Release Maintenance

For now, this tap is updated manually when a new Whale release is published.
Update `Formula/whale.rb` to point at the latest release tarballs and sha256
values from the Whale release assets.
